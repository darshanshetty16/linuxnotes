<strong> System Information <strong> <br>
Commands used to retrive data about the hardware,kernel and current system status <br>

<strong>uname</strong> <br>
<pre> uname -a </pre>
Prints detailed system information including the kernel version and processor architecture <br>

<strong>uptime</strong> <br>
<pre> uptime </pre>
Shows how long system has been running, the number of users,and system load averages<br>

<strong> date </strong> <br>
<pre> date </pre>
displays the current system date and time <br>

<strong> cal </strong>
<pre> cal </pre> 
Displays the simple text based calander in the terminal <br>


<strong> whoami </strong>
<pre> whoami </pre>
Prints the username of the current user <br>

<strong> who </strong>
<pre> who </pre>
Displays the a list of users who are currently logged into the system <br>

<strong> id </strong>
<pre> id </pre>
Prints the real and effective user and id group IDs for current user <br>
