1.<strong> File Viewing </strong> <br>
Commands used to display the contents of the file to the standard output in the Terminal screen <br>

<strong> cat </strong> <br>
<pre> cat filename </pre>
Concatinates and display the Entire content of a file in the terminal Screen <br>

<strong> tac </strong> <br>
<pre> tac filename</pre>
Displays the content file in the reverse order,starting from the last line.<br>

<strong> less </strong> <br>
<pre> less filename </pre>
Opens the file for interactive viewing ,allowing for forwad and backward navigation without loading the entire file <br>

<strong> more </more> <br>
<pre> more filename </pre>

<strong> head </strong> <br>
<pre> head filename </pre>
Outputs the first 10 line of the files by default <br>

<strong> tail <strong> <br>
<pre> tail filename </pre>
Outputs the last ten lines of the file by default <br>

<strong> tail -f </strong> <br>
<pre> tail -f filename </pre>
Follows a file in real time appending new data to the output as the file grows <br>

2.<strong> Text Analysis and Statistics</strong> <br>
Tools used to to count elements or provide structural information about text files <br>

<strong> nl </strong> <br>
<pre> nl filename </pre>
Gives the number of lines in a file <br>

<strong> wc </strong> <br>
<pre> wc filename </pre>
prints the newline,word, and byte counts for file <br>

3.<strong>Text Manuplation and sorting </strong> <br>
Commands used to reorganize or filter the data within file <br> 

<strong> sort </strong> <br>
<pre> sort filename </pre>
Arranges the lines of text file in alphabetical or numerical order <br>

<strong> uniq </strong> <br>
<pre> uniq filename </pre>
filters out adjacent duplicate lines from a sorted file to show only unique entries <br>
